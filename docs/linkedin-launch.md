I built `diagrammer`: a tiny Python tool that turns an LLM-written JSON spec into a clean blueprint-style SVG diagram.

The workflow is simple: describe the diagram in plain English, have an LLM emit the spec, run one command, and drop the SVG into a blog post or docs page. No design tool. No runtime dependencies. Stdlib-only Python.

It ships as:
- a CLI: `pipx install diagrammer`
- a Python API: `render(spec) -> svg_string`
- a Claude Code skill: `/plugin install diagrammer@diagrammer-tools`
